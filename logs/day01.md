# 学習メモ

## [JavaScript Primer](https://jsprimer.net/) を利用してJavaScriptの基礎を学習しなおし中
- JavaScriptやブラウザのAPIについては[MDN Web Docs](https://developer.mozilla.org/ja/)を参照するのが良い
- JavaScript Primerには[NotebookLM](https://notebooklm.google.com/notebook/295f7def-145e-4393-9e06-afb4e6e23781)が用意されている
- `use strict`という実行モードが存在し、厳格なエラーチェックが行われる
- 変数の宣言
  - `const` : 再代入できない変数を定義する。
  - `let` : 再代入可能な変数。初期化しなかった場合の値は`undefined`
  - `var` : `let`や`const`と違い、同じ名前の変数を再定義できてしまう。利用を避けたほうがいいケースが多そう。
- REPL = Read-Eval-Print Loop
