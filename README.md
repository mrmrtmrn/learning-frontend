# Frontend Learning Journey

フロントエンド技術（JavaScript/TypeScript/React）の体系的な学習記録

## 学習目標

### Phase 1: JavaScript 基礎（Week 1-2）
- 変数、関数、オブジェクト、配列
- 非同期処理（Promise, async/await）
- モジュール（import/export）
- クロージャ、this
- map, filter, reduce

### Phase 2: TypeScript 基礎（Week 3）
- 型の基礎（string, number, boolean, array）
- interface, type
- ジェネリクス（`<T>`）
- 型アサーション（`as`）

### Phase 3: React 基礎（Week 4-5）
- コンポーネント、JSX
- useState, useEffect
- props、イベントハンドリング
- 公式チュートリアル（三目並べ）

## 学習リソース

- [JavaScript Primer](https://jsprimer.net/) - JavaScript基礎
- [サバイバルTypeScript](https://typescriptbook.jp/) - TypeScript基礎
- [React 公式ドキュメント](https://ja.react.dev/) - React基礎
